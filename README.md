# Análisis de Ecommerce — Olist

Proyecto de análisis de datos de un curso de Data Analytics, usando el dataset público de Olist, plataforma brasileña de ecommerce.

## Objetivo
Analizar el desempeño comercial de Olist identificando métricas clave del negocio, comportamiento de inventario, principales vendedores y distribución geográfica.

## Herramientas
- Microsoft Excel
- Power BI
- Draw.io

## Fuente de Datos
Dataset público de Olist disponible en Kaggle.
Los archivos de datos no se incluyen en este repositorio por su tamaño.
